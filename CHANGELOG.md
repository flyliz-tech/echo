# Changelog

All notable changes to this project are documented here. This project follows [Semantic Versioning](https://semver.org/) and [Keep a Changelog](https://keepachangelog.com/).

## [0.0.1] - 2026-06-16

Initial public beta (Android).

### Added
- Offline-first task storage with SQLite (no account required).
- Time reminders with local notifications.
- Location reminders via geofencing with adjustable radius.
- Interactive map location picker with place-search autocomplete and full-screen mode.
- Map tab showing location tasks, with recenter-on-current-location.
- Calendar view for time-triggered tasks.
- Home list with sort modes, search, quick-complete, and multi-select delete.
- 10-second undo on delete.
- Light and dark themes with a consistent design-token system.

### Notes
- Android only; distributed as a sideloadable, debug-signed APK.
- Geofencing should be tested on a physical device.
