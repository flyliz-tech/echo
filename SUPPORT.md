# Support

## Getting Help

- Usage questions: open a GitHub Discussion in the `Q&A` category
- Bug reports: use the bug report issue template
- Feature ideas: use the feature request issue template

## Before Opening a New Issue

- Check existing issues/discussions for duplicates
- Include device model, Android version, app version, and reproduction steps
- Attach screenshots or logs when possible

## Security Issues

For vulnerabilities, follow `SECURITY.md` and use private reporting channels.
Do not post exploit details in public issues.
