# Echo User Guide

Echo helps you remember things at the right **time** or in the right **place**. This guide walks through the main features.

## First Launch

After installing, grant the permissions Echo asks for:

- **Location** (allow all the time) — needed for location reminders to trigger in the background.
- **Notifications** — needed to alert you when a reminder fires.

Without these, time/location reminders cannot notify you.

## Creating a Task

1. Tap the **+** tab.
2. Enter a **title** (required) and optional **notes**.
3. Add one or both triggers:
   - **Time reminder**: toggle it on and pick a date and time. You'll get a notification when it's due.
   - **Location reminder**: toggle it on, then search for a place or tap the map to drop a pin, and adjust the trigger **radius**. You'll be reminded when you arrive within that radius.
4. Tap **Save**.

> A task can have a time trigger, a location trigger, or both.

## The Map Picker

When adding a location reminder:

- Use the **search box** for autocomplete place suggestions.
- Tap anywhere on the map to move the pin; the place name updates automatically.
- Use the **+ / −** controls to change the trigger radius.
- Tap the **enlarge** button to open a full-screen map for finer placement.

## Home

- Your tasks are listed here. Tap a task to view it.
- **Search** by title, location, or notes.
- **Sort** with the toggle: All, By trigger time, By location, or Completed.
- **Complete** a task by tapping its checkbox. A 10-second **undo** appears after delete.
- **Multi-select**: long-press a task to enter selection mode, then select multiple (or select all) and delete them together.

## Map Tab

- See all your active location tasks as pins.
- Tap a pin to preview the task and open it.
- Tap **recenter** (the locate button) to zoom to your current location anywhere in the world.

## Calendar Tab

- Browse months; days with time-triggered tasks show a dot.
- Tap a day to see its scheduled tasks below.

## Viewing and Editing

- Open a task to see its details, including created/modified timestamps.
- Tap **Edit** to change it. Completed tasks can't be edited — mark them active again first.

## Settings

- Switch the appearance between system, light, and dark themes.

## Tips

- Test location reminders on a **physical device**; emulators are unreliable for geofencing.
- Keep background location enabled so reminders fire when the app is closed.
- If reminders stop firing, check that Echo is excluded from aggressive battery optimization.
