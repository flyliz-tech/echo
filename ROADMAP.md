# Roadmap

## Near Term (v0.0.x beta)

- Stabilize geofencing reliability on physical Android devices
- Improve reminder scheduling edge cases (timezone/daylight changes)
- Expand CI reliability and tighten quality gates
- Improve onboarding docs and first-time contributor experience

## Mid Term (v0.1.x)

- Release signing and repeatable release process
- Crash and performance telemetry evaluation (privacy-aware)
- Better task filtering and productivity UX improvements

## Long Term (v1.x+)

- Play Store distribution
- iOS support
- Optional authenticated sync architecture

Roadmap can change based on community feedback, bug severity, and maintainer capacity.
